About This Project

This project demonstrates key Data Structures and Algorithms (DAA) concepts applied to real-world scenarios:

  • DAA Input Enhancement: Articles are indexed using a hash table based on topics, enabling O(1) lookup time instead of O(n) linear search.
  • Space-Time Tradeoff: We use additional memory (cache) to store frequently accessed data, trading space for faster access time.
  • LRU Page Replacement: When the cache is full, the Least Recently Used item is evicted, mimicking operating system memory management algorithms.
  • Cache Optimization: The system tracks cache hits and misses to demonstrate the effectiveness of caching strategies.
